# WordMaster - 猜词小游戏集合

🎮 移植自 nonebot-plugin-wordle 和 nonebot-plugin-handle 的 AstrBot 游戏插件，支持英文猜单词、根据释义猜成语、多人对战、排行榜等功能。

## ✨ 功能特性

- 🔤 **猜单词** - 经典的英文猜单词游戏
- 🀄 **猜成语** - 根据释义猜四字成语
- ⚔️ **多人对战** - 支持多人同时参与，先猜中者获胜
- ⏱️ **限时模式** - 5分钟时间限制
- 📊 **排行榜系统** - 全服排名，展示顶尖玩家
- 📈 **个人战绩** - 记录胜率、连胜、最佳猜测等数据
- 💡 **智能提示** - 显示已排除字母、已确定位置
- 📖 **成语释义** - 猜中后显示成语含义
- 📚 **开源词典** - 使用 NLTK 英文单词库、chinese-xinhua 成语库

## 🚀 安装

### 方式一：通过 AstrBot 插件市场安装

1. 打开 AstrBot 管理面板
2. 进入「插件」→「插件市场」
3. 搜索 `WordMaster` 并安装

### 方式二：手动安装

1. 克隆仓库到插件目录：
   ```bash
   cd /path/to/astrbot/data/plugins
   git clone https://github.com/NumInvis/astrbot_plugin_wordmaster.git
   ```

2. 重启 AstrBot

### 使用示例

#### 1️⃣ 猜单词
```
/猜单词 6
```
开始一个6个字母的单词猜测游戏。

#### 2️⃣ 猜成语
```
/猜成语
```
根据显示的释义猜测四字成语。

#### 3️⃣ 多人对战
任何人都可以直接参与猜测，先猜中者获胜！

### 🎮 游戏规则

#### 猜单词 - 英文猜单词
- 猜一个指定长度的英文单词
- 🟩 **绿色** = 字母正确且位置正确
- 🟨 **黄色** = 字母存在但位置错误
- ⬜ **灰色** = 字母不存在
- 共 **6** 次机会
- 输入不符合规则的内容会被静默忽略，不消耗次数

#### 猜成语 - 根据释义猜成语
- 根据显示的**释义**猜测四字成语
- 🟩 **绿色** = 字正确且位置正确
- 🟨 **黄色** = 字正确但位置错误
- ⬜ **灰色** = 字不存在
- 共 **10** 次机会
- 输入非四字成语会被静默忽略，不消耗次数

#### 多人对战
- 任何人都可以直接参与猜测
- 先猜中者获胜
- 自动加入游戏，无需发送"加入"

### 💡 游戏中指令

- `提示` / `hint` - 获取提示
- `结束` / `退出` / `quit` - 结束游戏（仅房主）

### 📊 战绩统计

插件会记录以下数据：
- 🎮 总游戏数
- 🏆 获胜/失败次数
- 📈 胜率
- 🔥 当前连胜/最高连胜
- 🎯 最佳猜测次数
- 📊 平均猜测次数
- ⏱️ 平均用时
