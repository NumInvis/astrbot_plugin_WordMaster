# WordMaster - 猜词小游戏集合

🎮 移植自 nonebot-plugin-wordle 和 nonebot-plugin-handle 的 AstrBot 游戏插件，支持英文猜单词、汉字猜成语、多人对战、排行榜等功能。

## ✨ 功能特性

- 🔤 **Wordle** - 经典的英文猜单词游戏
- 🀄 **Handle** - 汉字版 Wordle，猜成语游戏
- ⚔️ **多人对战** - 支持多人同时参与，先猜中者获胜
- ⏱️ **限时模式** - 增加时间限制，挑战你的反应速度
- 📊 **排行榜系统** - 全服排名，展示顶尖玩家
- 📈 **个人战绩** - 记录胜率、连胜、最佳猜测等数据
- 💡 **智能提示** - 显示已排除字母、已确定位置
- 📖 **成语释义** - 猜中后显示成语含义
- 📚 **多词典支持** - CET4/CET6/GRE/IELTS/TOEFL 等多种英文词典

## 🚀 安装

### 方式一：通过 AstrBot 插件市场安装

1. 打开 AstrBot 管理面板
2. 进入「插件」→「插件市场」
3. 搜索 `WordMaster` 并安装

### 方式二：手动安装

1. 克隆仓库到插件目录：
   ```bash
   cd /path/to/astrbot/data/plugins
   git clone https://github.com/NumInvis/astrbot_plugin_wordmaster.git
   ```

2. 重启 AstrBot

### 使用示例

#### 1️⃣ 单人猜单词
```
/wordle CET4 5
```
开始一个CET4词典、5个字母的单词猜测游戏。

#### 2️⃣ 限时模式
```
/wordle CET6 6 --timed
```
6个字母的限时挑战，5分钟内必须猜出！

#### 3️⃣ 多人对战
```
/wordle对战 CET4 5
```
创建对战房间，其他玩家发送"加入"即可参与。

#### 4️⃣ 猜成语
```
/handle --strict
```
开启严格模式，猜测必须是有效的四字成语。

### 🎮 游戏规则

#### Wordle - 英文猜单词
- 猜一个指定长度的英文单词
- 🟩 **绿色** = 字母正确且位置正确
- 🟨 **黄色** = 字母存在但位置错误
- ⬜ **灰色** = 字母不存在
- 共 **6** 次机会

#### Handle - 汉字猜成语
- 猜一个**四字成语**
- 🟩 **绿色** = 正确
- 🟨 **黄色** = 存在但位置错误
- ⬜ **灰色** = 不存在
- 每个格子显示：**汉字 + 声母 + 韵母 + 声调**
- 共 **10** 次机会

#### 限时模式
- 增加 **5分钟** 时间限制
- 超时自动结束游戏
- 挑战你的反应速度！

#### 多人对战
- 先猜中者获胜
- 其他玩家发送 **"加入"** 参与对战
- 房主发送 **"开始"** 开始游戏
- 支持多人同时猜测

### 💡 游戏中指令

- `提示` / `hint` - 获取提示（显示已排除字母、已确定位置）
- `结束` / `退出` / `quit` - 结束游戏

### 📊 战绩统计

插件会记录以下数据：
- 🎮 总游戏数
- 🏆 获胜/失败次数
- 📈 胜率
- 🔥 当前连胜/最高连胜
- 🎯 最佳猜测次数
- 📊 平均猜测次数
- ⏱️ 平均用时
